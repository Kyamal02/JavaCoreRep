1. Реализовать класс BoundedBuffer.
2. Работа с данной коллекцией должна быть потокобезопасной.
3. Если какой-то поток захочет получить элемент из пустого буфера, то он должен будет ждать пока элемент в буфере не появиться.
4. Если какой-то поток захочет положить элемент в полный буффер, то он должен ожидать пока не появиться место для этого.
